Gmail Spam Detector using Naive Bayes Classification model and Google Cloud API

GitHub Repository

Using Scikit Learn's naive bayes classification library I created a machine learning model that can detect spam emails using term frequency-inverse document frequency with a 96% accuracy.

I then used Google Cloud's Gmail API to deploy the model. The program scans all unread emails in the users inbox and labels them as spam or not spam, placing spam emails in the users spam folder.
